# 关闭防火墙

```shell
systemctl stop firewalld.service
systemctl disable firewalld.service
yum install iptables-services 
service iptables start
service iptables stop
```

# 挂载硬盘

```shell
echo /dev/vdb1 /mnt ext3 defaults 0 0 >> /etc/fstab
mount /dev/vdb1 /mnt
df -hl
```
