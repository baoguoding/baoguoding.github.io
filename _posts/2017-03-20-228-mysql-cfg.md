# 自定义mysql tmp目录

## 错误
    /usr/sbin/mysqld: Can't create/write to file '/tmp/ibZ9guDH' (Errcode: 13 - Permission denied)

## 更新临时目录
    /etc/my.cnf
    [mysqld]
    tmpdir=/usr/local/tmp

## 添加权限
    chown root.root /usr/local/tmp -v
    chmod 1777 /usr/local/tmp -v
