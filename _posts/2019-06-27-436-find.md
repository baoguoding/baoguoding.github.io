# 按照文件大小查找文件

```shell
find / -type f -size +20M  -print0 | xargs -0 du -h | sort -nr
```


## type 查找某一类型的文件

+ b 块设备文件。
+ d 目录。
+ c 字符设备文件。
+ p 管道文件。
+ l 符号链接文件。
+ f 普通文件

## size

+ b 块（512字节）
+ c 字节
+ w 字（2字节）
+ k 千字节
+ M 兆字节
+ G 吉字节

# 示例2

```shell
find -H .                      \
	\(                         \
	-not \(                    \
		\(                     \
			-path ./vendor -o  \
			-path ./_\* -o     \
			-path ./.\* -o     \
			-path ./docs       \
		\) -prune              \
	\)                         \
	\)                         \
	"$@"                       \
	| sed 's|^./staging/src|vendor|'
```

find -H . (-not((-path ./vendor -o -path ./_\* -o -path ./.\* -o -path ./docs) -prune)) "$@" | sed 's|^./staging/src|vendor|'
